#Flux Boilerplate

###Info:

I've made this Flux Boilerplate while i was practicing the Flux Architecture, it will come in handy for future use.

###Frameworks & Languages used:

- Flux
- ReactJS
- NodeJS
- JavaScript (ES6, ES2015)
- HTML5/CSS3
- JSON
- jQuery
- AJAX
- Bootstrap
- Gulp
- Gulp-open
- Gulp-connect
- Browserify
- Babelify


###Installation

```
$ npm install

$ gulp

$ open http://localhost:3070

```
